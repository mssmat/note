Notionでnote風のブログを作ってみた

作成日時2025-03-24 21:53 TagGitHub, Notion

みんな大好きNotionをヘッドレスCMS的に使ったnote風のブログを作った

書くのはNotionで。
ホスティングはGitHubで。
一応どちらも無料。有料プランもあるけど。

APIでNotionの記事をブロックごとにゲッツ。
それを静的なHTMLに変換しコミット、APIでGitHubにプッシュ。

実は自宅サーバーでやっていましたが、色々と考えることがあり、GitHubに移行しました。
独自ドメインも使えるらしい。

何よりNotionで書けるのがラク。管理もラク。
画像貼ったり、動画貼ったり、ツイート貼ったり。
思いついたらすぐ書ける。スマホからも更新できる。

弱点は、Notionで保存し、AWSにホスティングする、払い出された画像などの署名付きURLの有効期間が1時間なこと。
つまりデプロイして1時間が経つと無効になるので画像などが表示されなくなります。
対策は、1時間たつ前に再度同じ手順でデプロイすればいいだけです。これは自動化できますね。

同じようなことをしている人は探すと、実は結構います。
Super.soとかペライチ(Wraptas)とか、色々な有料サービスもあります。

画像や動画

作成日時2025-03-24 22:32 TagNotion

画像や動画などを直接Notionに保存した場合、クラウドの保存先のURLの有効期限は1時間。
時間が過ぎると画像が表示されなくなります。

これを防ぐには、ちょくちょく更新してURLを更新する必要があります。

Notionだと画像を貼るのもラクなので便利です。

動画も保存できる。

        function isPrime(num) {
            if (num <= 1) return false;
            if (num <= 3) return true;
            if (num % 2 === 0 || num % 3 === 0) return false;
            for (let i = 5; i * i <= num; i += 6) {
                if (num % i === 0 || num % (i + 2) === 0) return false;
            }
            return true;
        }

        let currentNumber = 2;
        function displayNextPrime() {
            while (!isPrime(currentNumber)) {
                currentNumber++;
            }
            const primeContainer = document.getElementById('prime-container');
            primeContainer.innerHTML = '';
            const primeElement = document.createElement('div');
            primeElement.className = 'prime-number';
            primeElement.textContent = currentNumber;
            primeContainer.appendChild(primeElement);
            let opacity = 0.2;
            const interval = setInterval(() => {
                opacity += 0.1;
                primeElement.style.color = `rgba(226, 232, 240, ${opacity})`;
                if (opacity >= 1) {
                    clearInterval(interval);
                    setTimeout(() => {
                        let fadeOutInterval = setInterval(() => {
                            opacity -= 0.1;
                            primeElement.style.color = `rgba(226, 232, 240, ${opacity})`;
                            if (opacity <= 0.2) {
                                clearInterval(fadeOutInterval);
                                currentNumber++;
                                setTimeout(displayNextPrime, 1000);
                            }
                        }, 100);
                    }, 1000);
                }
            }, 100);
        }

        displayNextPrime();

import time

def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True

def prime_generator():
    num = 2
    while True:
        if is_prime(num):
            yield num
        num += 1

def display_primes():
    for prime in prime_generator():
        print(prime)
        time.sleep(1)

if __name__ == "__main__":
    display_primes()
