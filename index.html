RocketChatを導入

作成日時2025-05-07 19:23 TagAI, VPS
Rocket.Chat | Secure CommsOS™ for Mission-Critical Operations
https://www.rocket.chat/

Slackの代わりに、自社ホスティングのRocketChatを導入した。
(Skype→LINE WORKS→RocketChat)

【条件】

  • ネット検閲のある某C国ユーザーとやりとりできること(without VPN)
  • ブラウザ、PCソフト、スマホアプリで使え、PCやスマホからシームレスに使えること

Skypeで特に困ってなかったのですが、5月にサービスが終了。
そこでLINE WORKSに移行した。高機能でかなり良かったが、無料だと容量が5GBまで。まあこれは当然で、仕方ない。
サブスク? MATさんはノーサンキューです、もちろんww

そこでRocketChatを導入することに。
RocketChatはSlackライクなOSSだそうです。
Slackは使ったことがないですが、チャレンジしてみよう。

オンプレミスだとハードの用意から必要になるので、VPSを借り、そこに展開する。
OSはubutuを選ぶ。
GUIも導入できるので、Linuxに慣れてなくても最悪何とかなるはず。

少しのコマンドですぐに使えるようになった。
(サーバーの初期設定などは除く)
しかも半自動で独自ドメインがTLSで使えるように。
SSL Labsでの評価も「A」。

先人たちには感謝しかない。

某C国からもアクセスできることは確認済み。
VPS費用は月に500円～1000円程度(CPUやメモリ、ストレージによる)。

分からないことがあれば、人に聞く、ググる、本で調べる。
これにAIが加わった。THE 日常使い。

全てのアトピー患者さんへ。どうか、この動画を「一番最初に」見てください。

OpenAI Responses APIを使う(Built-in toolsとreasoning summary)

作成日時2025-04-18 21:08 TagAI

o4 miniのAPIを使って、Responses APIを試してみたがうまくいかない箇所があった。

先に結論

新しいResponses APIは、toolsの書き方なども、Chat Completions APIから大きく変わってしまい、既存のコードを大幅に変更しないといけない。
非常に面倒くさい。
Chat Completions APIの拡張ではダメだったのだろうか..
AIに相談して決めて欲しかったww

問題①

Built-in toolsでは「web_search」しか使えなかった。
「code_interpreter」はエラーになった。2025年4月18日時点ではどうやらまだ使えないらしい。
「file_search」は試してない。

問題②

Built-in toolsと自作のFunction Callingをtoolsで同時に設定すると、プロンプト内容に関わらずなぜか常にFunction Callingが実行される。

問題③

reasoningのsummaryをリクエストに加えるとエラーになる。推論過程を見たいのだがどうすればいいのだろう?

json 
"reasoning": {
"effort": "medium"
}

これはOK。

json 
"reasoning": {
"effort": "medium",
"summary": "detailed"
}

これはダメ。

例

以下のリクエストだとエラーにはならないが、Function Callingが呼び出される。なぜかBuilt-in toolsでweb_searchが呼び出されない。
tool_choiceもauto、strictもfalseで、プロンプトもあからさまなのに何でかな?

bash 
curl -X POST \
  https://api.openai.com/v1/responses \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ＜＜Your OpenAI API Key＞＞" \
  -d '{
    "model": "o4-mini-2025-04-16",
    "input": [
      {
        "role": "developer",
        "content": [
          {
            "type": "input_text",
            "text": "あなたは博多弁で喋ります"
          }
        ]
      },
      {
        "role": "user",
        "content": [
          {
            "type": "input_text",
            "text": "最新の日経平均株価をサーチで調べて"
          }
        ]
      }
    ],
    "text": {
      "format": {
        "type": "text"
      }
    },
    "reasoning": {
      "effort": "medium"
    },
    "tool_choice": "auto",
    "tools": [{"type": "web_search", "search_context_size":"medium", "user_location": {"type": "approximate", "city": "Fukuoka", "country": "JP"}}, {"type": "function", "name": "get_city_names", "description": "街の名前が出た場合にのみ動作。街の名前を全て配列で出力する。", "parameters": {"type": "object", "properties": {"cities": {"type": "array", "items": {"type": "string", "description": "街の名前"}}}, "additionalProperties": false, "required": ["urls"]}, "strict": false}],
    "store": false,
    "stream": true
  }'
